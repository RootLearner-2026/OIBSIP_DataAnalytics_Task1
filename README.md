# Exploratory Data Analysis on Retail Sales Data

## Oasis Infobyte Internship

**Domain:** Data Analytics
**Task:** Task 1

---

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a retail sales dataset to identify patterns, trends, and customer purchasing behavior.
The analysis helps businesses understand product demand and sales distribution using statistical analysis and visualization techniques.

---

## 🎯 Objective

The objective of this project is to analyze retail sales data and extract meaningful insights by performing exploratory data analysis and creating visualizations.

---

## 📊 Dataset Description

The dataset contains information about retail transactions including customer demographics and purchase details.

### Dataset Features

* **Transaction ID** – Unique identifier for each transaction
* **Customer ID** – Unique identifier for customers
* **Gender** – Customer gender
* **Age** – Customer age
* **Product Category** – Category of product purchased
* **Quantity** – Number of items purchased
* **Price per Unit** – Price of each product
* **Total Amount** – Total value of the purchase

---

## 🛠 Tools & Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## ⚙️ Steps Performed

1. Data loading using Pandas
2. Data inspection and structure analysis
3. Checking for missing values
4. Statistical summary of dataset
5. Exploratory data analysis
6. Data visualization using Matplotlib and Seaborn

---

## 📈 Data Visualizations

The following visualizations were created:

* Sales Distribution Histogram
* Gender-based Sales Analysis
* Product Category Distribution
* Customer Age Distribution

These visualizations help understand customer purchasing patterns and product demand.

---

## 🔍 Key Insights

* Certain product categories have higher purchase frequency.
* Customer purchases vary across different age groups.
* Gender-based purchasing behavior can be observed.
* Most transactions occur within a specific sales range.

---

## 📚 Conclusion

Exploratory Data Analysis helped identify patterns and trends in retail sales data.
The insights generated from this analysis can help businesses improve product strategies, customer targeting, and inventory management.
